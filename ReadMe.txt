Обязательно перед запуском прописать Alias("ParserZP") для базы данных (db\database.mdb).

Инструкция создания Alias:
Пуск->Панель управления->Администрирование->Источники данных (ODBC);
или
В командной строке "odbcad32";
Пользовательский DSN->Добавить;
Выбрать драйвер "Driver do Microsoft Access (*.mdb)";
Имя источника данных "ParserZP";
База данных->Выбрать, указать полный путь в базе данных (db\database.mdb).
